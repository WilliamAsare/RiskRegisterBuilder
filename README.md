## Features

- **Risk Heatmap** — interactive 5×5 matrix with inherent/residual toggle and forecast overlay
- **CIA Triad Assessment** — questionnaire-driven asset criticality ratings
- **Investment Forecasting** — track planned security spend with projected residual scores
- **Framework Mapping** — NIST CSF 2.0, ISO 27001, SOC 2, and ITGC templates
- **Control-Threat Linking** — map controls to the specific threats they mitigate
- **Collaboration** — share registers with teammates as viewers or editors
- **Export** — PDF reports and Excel workbooks
